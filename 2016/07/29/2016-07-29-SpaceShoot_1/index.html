一步一步构建“太空射击”Unity3D游戏（一）

发表于 2016-07-29   |   分类于 Unity3D教程   |     |  

这篇教程是我上学的笔记，Unity3D初学，一步一步从零开始构建一个“太空射击”游戏。

一、游戏功能概述

一个简单的太空射击游戏，将实现自动出现陨石、敌机，玩家需要控制自己的飞机躲避陨石，或者击落敌机。

二、开发环境与素材

Unity3D，我的版本是，Unity3D 5.1.2f 64位 Mac
素材SpaceShooter.unitypackage，网盘下载链接：网盘，密码: p2p5

三、资源导入

  1. 新建工程，格式为3D格式，确认自己的保存路径；
  2. 点击Unity菜单栏Assents -> Import package -> custom package；
  3. 选择准备的素材SpaceShooter.unitypackage，导入即可；

四、创建玩家飞机

  1. 在Project搜索栏中找到vehicle_playerShip，拖拽至Hierarchy；

  2. 调整摄像机的视角，使PlayerShip在视线正下方；

  3. 给飞机添加碰撞
    添加MeshCollider选中vehicle_playerShip，点击Inspector中的Add Componet，找到Mesh Collider，添加player_ship_collider，勾选覆盖，则自动将碰撞的范围确定为飞机自身。
    使用简易的模型替换复杂模型实现碰撞效果：

    覆盖前：

    覆盖后：

  4. 设置触发器选项，勾上Mesh Collider -> is Trigger，模型在于陨石发生碰撞时候，要发生爆炸，需要起到一个触发的作用。

  5. 设置刚体，为玩家飞机加上Rightbody

  6. 取消重力，否则飞机会往下掉，去除Use Gavity的勾

  7. 添加尾焰效果
    在Project搜索栏中找到engines_player，拖拽至Playership。
    飞机模型变为：

点击运行之后，飞机变为：

  1. 将该飞机保存为预置体
    在Hierarchy窗口中，拖拽Player_ship到Project窗口/Assents/Prefabs文件夹即可

五、玩家飞机材质（Standard）和灯光

  1. 调整main_light灯光位置、角度，该光源的强度可以大一些；
  2. 新建一个名为fill_light的平行光，该光源的强度可以较小；

  3. 新建一个从下往上的灯光Down_light，目的为把物体的轮廓表示出来;
    效果如图：

  4. 调整飞机材质、贴图
    此处先选用在Metal上使用Stander，在glass上使用Stander（Specular setup）
    两个材质球功能很强大，可以自行调整参数，进行更多的摸索。
    如图：

六、添加太空背景

  1. 在Hierarchy中创建一个Quard，修改名字为BackgroundMap

  2. 调整方位与大小

  3. 添加图片

  4. 调整图片比例，将X与Y的比例调整为1:2，得到不拉伸的图。

  5. 调整相机的位置和角度，使整个页面的大小位置合适，距离飞机的高度也合适
