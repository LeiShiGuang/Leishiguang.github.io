“初窥”项目管理03 - 鸟瞰IT项目管理

发表于 2016-07-28   |   分类于 “初窥”项目管理   |     |  

“让人们做事的艺术” —— 玛丽·法雷特（Mary Follett），二十世纪早期管理学作家
“世事洞明皆学问，人情练达即文章”——《红楼梦》

在现实当中，离不开人和事。而管理就是，让人和事整合得很好。
思想：管理管理，管的是事，理的是人

管事：事情是死的，有规律的，在管事则有方法、技术、工具。
理人：梳理人与人之间的关系，“你怎么老是不理我啊”，这个理就是沟通。

项目管理

Project Mangement，将知识、技能、工具与技术应用于项目活动，以满足项目的要求。

上图显示了项目管理分为管事、理人，与整合的关系。

  • 管事有许多的方法与技能
  • 理人有许多的理论与经验
  • 整合，便是让人做事的艺术

什么是项目

项目的三大特性：

  • 独特性：历史上不存在完全相同的项目
  • 临时性：有明确的起止时间，即便是有些项目时间很长
  • 渐进明细：在项目早期刚过开始的时候，有许多的未知，很多的不可控，也只有在项目的逐渐展开中，通过计划和控制，渐进明细。

所谓的项目管理，就是把渐进明细做好。项目经历的各项活动，都是为了如何让渐进明细做得更加准确。

任何项目都有这三个特征。
任何有这三个特征的，都可以当做项目。可以用项目管理的方法来管理…（搞生产和搞运营，都不是项目管理….）

比如：修建一栋居民楼…
盖一栋楼，独特性明显不明显…（有太多太多的未知性，独特！）
临时性，固定的起止时间
渐进明细…从图纸的绘制开始，渐渐明确地质之类的….

比如2：卖的很好的一个汽车，再生产一万两….
是否有独特性…没有
渐进明细是否，没有。
不是一个项目..

最后，一句话：
项目管理的很多思想、很多方法，都是围绕着确保过程和结果是成功的。
