吃饭是大问题

发表于 2017-04-23 | 分类于 个人念想 |

出门在外，打破以往的饮食习惯，难以寻求到其它替代，我因此而悲伤难过。

盖浇个饭，一饭一菜干和干拌，寡淡无味。
刀削碗面，没荤没素全是红油，营养失衡。
大鱼大肉，却也油腻。
不如啃个馒头，抓把香蕉，叼块鸡腿，拿枚鸡蛋来得自在。
出门在外，身体健康着实重要…

同一店子多吃要腻，徘徘徊徊食之无味弃之可惜。
不同店子参差不齐，忐忐忑忑下面一家会好吃吗？
店子繁多，门里门外难辨深浅。
出门在外，处世经验着实重要…

人们总有许多追求，有了馒头想吃包子，有了美味还要佳肴。
新鲜复新鲜，刺激复刺激，何时是头难以自知。
麻痹的神经，会不会回忆往昔空空落泪？
出门在外，知足饱满着实重要…

吃饭真乃大问题。
自己做饭不失为一出良计，私人定制健康无忧。
但大好青春每一刻都很值钱的时候，花太多时间自己做饭，恐怕还是需要些勇气。
