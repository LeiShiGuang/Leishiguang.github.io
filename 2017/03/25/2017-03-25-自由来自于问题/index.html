自由来自于问题

发表于 2017-03-25 | 分类于 个人念想 |

“求实求真”早已包含在我本科学校的校训之中，但是什么才是真实的呢？
你怎么知道，你现有的信息和知识，是不是真的？

在不断认识自己的人生道路上，我遇到过许多拦路虎，这一只尤为凶险。
稍不留神，已有的价值观念即被颠覆，一切的智识体系均需重建。

你该相信什么，真理在哪里呢？

生活在安逸圈，以为很自在，却看不到是被绳索束缚。
不断接受知识，以为很充实，却认不清是为潮流戏弄。

你在寻找什么？你现有的信息和知识，是不是真的？

经常朝三暮四，妄自生出许多观念，但难以认清自己的。
受着周边的耳濡目染，接受思想的熏陶洗礼，也始终不辨真假。

于是，我们已知的事物，不能让我们得到平静。
只有未知的那些，才能帮助我们更加认识自己。
将误以为是“知识”的答案都抛开，提出问题、超脱已知事物，才能得到无限伸展的自由。

了解自己的过程，从问问题开始。
除了你自己，没有人能帮助你。

By 雷世光
