美好的世界在心里

发表于 2017-03-18 | 分类于 个人念想 |

我们真正需要的，没人能给，也没有人能让我们快乐。我们要做的，就是直面自己内心深处的需求。
往往，与他人发生争执，其根本原因并非在于他人，而在于自己。
自己的不开心、失望和愤恨，来自于期望的破灭。这些期望，常来自于内心深处的需求。

深藏在内心深处的需求，可能是小时候的心理阴影，可能是大众的潜移默化，可能是过去未解决的伤痛。
比如我，童年时候的“祈愿”经常遭到数落。这样的种子，种在内心深处，偶尔想起，就深切感到“自己不被重视”。
每当情绪冲动的事情发生时，童年的想法便萦绕不去：你这么做，就是不重视我。

举个例子，你的恋人忘了你的生日，为此你感到愤恨。
其根本原因真的就是你希望恋人记得生日，而并非希望恋人证明他们重视自己吗？
实际想想，为何需要通过“记得生日”证明“重视自己”呢？
总是这么抱着怀疑，便感觉真是这样一般。

期望的目的是要让需求得到满足，但这同时，必然会发生两件事：
其一，感到内心的空洞，假如感觉自己需要某样东西，一定以为我们没有。
其二，如果没有某人来满足这份需求，我们就会觉得失望与愤恨。

“通往地狱之路，是由期望铺成的”。
期望会把接受和让人自由等充满爱意的感觉，挡在门外。
越来越多的期望，便越来越感受不到内心的声音。

对周遭世界，也是如此。总是抱着“世界应当这样”的怀疑，忽视心中的爱，便难以感受世界美好。
自然，外边世界充满险恶，焦躁不安的生活方式、喘不过气的高额房价、漂泊不定的浮躁人心……
这些要看不到吗？相反，应当正视这些。

蒙蔽自己的双眼，无法在前行的熙熙攘攘中看清道路。
封闭内心的声音，终将在人生的十字路口上迷失方向。

心，早已拥有了一切。

By 雷世光
